CSS TABLE METHOD

Родительский div отображается в виде таблицы, дочерний div отображается в виде ячейки. div-ячейке устанавливается вывод содержимого по-середине

Плюсы:
  • Хорошо подходит для динамического содержимого popup
Минусы:
  • Не работает в ранних версиях IE
  • Требуется дополнительный HTML тэг
JS SET TOP AND LEFT

В сценарии JS div-элементу устанавливаются свойства top и left

Плюсы:
  • Подходит для динамического содержимого
Минусы:
  • Центрируется средствами JS

* Для возвышения над другими элементами используется min-height

ABSOLUTE POSITIONING AND STRETCHING

Расстояния от всех четырех краев div-элемента до границ браузера устанавливается в 0. Свойство margin: auto расположит div-элемент в центре браузера

Плюсы:
  • Не выявлены
Минусы:
  • Не работает в ранних версиях IE
  • Надо указывать высоту div-элемента, иначе растягивается по всей высоте браузера
ABSOLUTE POSITIONING AND NEGATIVE MARGIN

Левый верхний угол div-элемента установить в центре браузера. В сценарии JS свойство margin подвинуть div-элемент влево вверх, что бы он оказался в центре

Плюсы:
  • Подходит для динамического содержимого
Минусы:
  • Центрируется средствами JS

Если знать размеры div-элемента, то можно обойтись без JS
